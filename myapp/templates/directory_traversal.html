📖 DOM-based XSS란?
서버를 거치지 않고 클라이언트(브라우저)의 JavaScript에서 URL 파라미터를 직접 DOM에 삽입할 때 발생하는 XSS 공격입니다. 서버 로그에도 남지 않아 탐지가 어렵습니다.

💡 취약한 코드 vs 안전한 코드

// ❌ 취약 (innerHTML 사용)
element.innerHTML = "안녕하세요, " + name + "님!";
// HTML 태그가 파싱되어 스크립트 실행

// ✅ 안전 (textContent 사용)
element.textContent = "안녕하세요, " + name + "님!";
// 모든 입력을 텍스트로만 처리
