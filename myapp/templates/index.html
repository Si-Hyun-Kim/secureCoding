보안 취약점 테스트

각 보안 취약점을 선택하여 테스트해보세요

💉
SQL Injection
데이터베이스 쿼리 조작을 통한 무단 접근 테스트
⚙️
Command Injection
시스템 명령어 실행을 통한 코드 삽입 테스트
📁
Directory Traversal
경로 조작을 통한 파일 시스템 접근 테스트
🔄
Reflected XSS
URL 파라미터를 통한 즉각적인 스크립트 실행 테스트
💾
Stored XSS
DB 저장을 통한 지속적인 스크립트 공격 테스트
🌐
DOM-based XSS
클라이언트 측 DOM 조작을 통한 XSS 공격 테스트

📌 테스트 안내

각 테스트는 취약한 코드와 안전한 코드 두 가지 모드를 제공합니다.
취약한 코드에서 공격이 성공하는지 확인하고, 안전한 코드에서는 어떻게 방어되는지 학습하세요.
